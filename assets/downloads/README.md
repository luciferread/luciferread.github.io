# README - Sample Chapter Files

## This folder is for your sample chapter downloads

Please add the following three files to this folder:

1. `the_radiants_sample.epub` - For most e-readers (Apple Books, Kobo, Nook, etc.)
2. `the_radiants_sample.pdf` - Universal format, works on any device
3. `the_radiants_sample.mobi` - For Kindle devices

## How to Create These Files

See the comprehensive guide: `newsletter_setup_guide.md` in the brain folder.

**Quick steps:**
1. Extract Chapters 1-3 from your manuscript (approximately 30 pages, ~10% max for KDP Select)
2. Use Calibre (free) to create EPUB and MOBI from your Word/Docs file
3. Export PDF from Word/Google Docs
4. Test each file before uploading
5. Copy files to this folder
6. Run: `git add assets/downloads/ && git commit -m "Add sample files" && git push`

## File Requirements

- **Maximum size**: Keep each file under 1MB for fast downloads
- **Filenames**: Must match exactly (case-sensitive):
  - `the_radiants_sample.epub`
  - `the_radiants_sample.pdf`
  - `the_radiants_sample.mobi`

## Testing

After uploading, verify your files are accessible:
