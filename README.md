# Lucifer Read's Author Website

A professional, high-performance static website built for sci-fi authors. This site is designed with a "dark mode" aesthetic, optimized for SEO, and built specifically for hosting on **GitHub Pages**.

## üöÄ Features
- **Sci-Fi Aesthetic**: Custom-styled with a dark, atmospheric theme inspired by sci-fi book covers.
- **Newsletter Ready**: Integrated with Substack for direct reader engagement.
- **Bookshelf**: Display your works with direct purchase links and upcoming release badges.
- **Direct Support**: Pre-configured (but hidden) support for BookFunnel and BookVault links.
- **Media/Press Kit**: Built-in section for bloggers and press to download high-res assets and bios.
- **SEO Optimized**: Includes `jekyll-seo-tag` and `jekyll-sitemap` for search visibility.

## üõ†Ô∏è Local Development

### Prerequisites
- Ruby (3.0 or higher)
- Bundler

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Start the local development server:
   ```bash
   bundle exec jekyll serve
   ```
4. Open your browser to `http://localhost:4000`.

## üåê Deployment
This site is pre-configured for **GitHub Pages**. To deploy:
1. Push your code to a GitHub repository.
2. In the repository Settings, navigate to **Pages**.
3. Set the source to the `main` branch (root folder).
